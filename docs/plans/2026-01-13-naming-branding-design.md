# Naming & Branding Design - OdFachowca

Data: 2026-01-13

## Naming

### Proces wyboru nazwy

1. **Analiza konkurencji:**
   - Polska: SCCOT, SYKAL, WINBUD (kosztorysowanie desktop), Fixly, Remontuj.pl (marketplace)
   - Zagranica: Housecall Pro, Jobber, ServiceTitan, Workiz (Field Service Management)
   - **Wniosek:** Brak mobile-first, quick quote app w Polsce

2. **Kierunki nazewnicze:**
   - Odrzucone: Quote-based (nie kojarzy się w PL z wyceną)
   - Odrzucone: Majsterio (konflikt z siecią sklepów Majster)
   - Wybrane: "Od Fachowca" - uniwersalne, buduje zaufanie

### Finalna nazwa

| Element | Wartość |
|---------|---------|
| **Nazwa** | OdFachowca |
| **Domena** | odfachowca.pl |
| **Koncept** | "Wycena od fachowca" - jasne dla klienta |

### Dlaczego OdFachowca działa

- **Uniwersalne:** Fachowiec = hydraulik, elektryk, stolarz, malarz, HVAC...
- **Zrozumiałe dla klienta:** Link `odfachowca.pl/q/abc` - od razu wiadomo co w środku
- **Buduje zaufanie:** Profesjonalne źródło, nie spam
- **Polskie:** Naturalne brzmienie

## Branding

### Vibe marki

**Solidny/Rzemieślniczy** + **Przyjazny/Przystępny**

Jak Carhartt ale dla software - solidna robota, godny zaufania, ale nie korporacyjny.

### Inspiracje

- **Helixcare** (Dribbble) - lekki granat, kremowe tło, nowoczesny healthcare
- **Claude** - terracotta accent, papierowe tło

### Paleta kolorów

| Rola | Nazwa | Hex |
|------|-------|-----|
| Primary | Żywy niebieski | `#3B5EDB` |
| Accent/CTA | Terracotta | `#DA7756` |
| Background | Kremowy papier | `#F8F6F3` |
| Surface | Biały | `#FFFFFF` |
| Heading | Ciemny granat | `#1A2B4A` |
| Body text | Szary | `#64748B` |
| Success | Teal | `#0EA5A0` |
| Warning | Amber | `#D69E2E` |
| Error | Czerwony | `#DC2626` |

### Logo

- **Typ:** Symbol + Wordmark
- **Symbol:** Zaokrąglona litera "F"
- **Wordmark:** "**Od**Fachowca" (Od w terracotta)
- **Styl:** Przyjazny, zaokrąglony

### Typografia

- **Font:** Nunito (Google Fonts)
- **Charakter:** Humanistyczny sans-serif, ciepły, przyjazny
- **Headings:** Nunito Bold/SemiBold
- **Body:** Nunito Regular

### UI Style

- **Zaokrąglenia:** Soft & rounded (12-16px)
- **Cienie:** Delikatne (0 4px 12px rgba(0,0,0,0.08))
- **Spacing:** 4px base grid
